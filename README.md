# Crimes-in-Medellin-Forecasting
Spatio Temporal forecasting on Neighbors and Districts of Medellin as DS4A / Colombia Cohort 6 final project.

## Datasets
The datasets used on this project comes from te web page of the "Alcaldia de Medellin", it can be found on http://medata.gov.co/node/85, as 34 datasets, that have the same structured but are related to diferent topics such as Theft, Capture, homicide, among others.
